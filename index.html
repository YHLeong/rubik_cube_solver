How to Use

  1. Select a color from the palette
  2. Click on cube squares to paint them
  3. Color all 54 squares (9 of each color)
  4. Click "Solve Cube" to get step-by-step solution
  5. Use "Previous" and "Next" to navigate solution steps

Solution Controls

  • Next/Previous: Step through moves manually
  • Play/Pause: Auto-play solution with animations
  • Auto: Fast solve without pausing
  • Speed Control: Adjust animation speed
